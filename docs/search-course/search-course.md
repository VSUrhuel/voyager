        <h2>Search Course</h2>  
<p>The Search Course feature allows users to quickly find courses by entering keywords in the search bar. This feature helps users efficiently browse available courses, filter relevant options, and access course details seamlessly.</p>

<h2>Use Case Scenario</h2>  
<table border="1">
    <tr>
        <th>Use Case</th>
        <th>Search Course</th>
    </tr>
    <tr>
        <th>Actor(s)</th>
        <td>Mentee</td>
    </tr>
    <tr>
        <th>Goal</th>
        <td>Allow users to search for available courses efficiently.</td>
    </tr>
    <tr>
        <th>Preconditions</th>
        <td>User must be a registered mentee and be on the course search page.</td>
    </tr>
    <tr>
        <th>Main Scenario</th>
        <td>
            1. User navigates to the "Courses Offered" page.<br>
            2. User enters a keyword in the search bar.<br>
            3. The system filters and displays relevant courses matching the search term.<br>
            4. If no results are found, the system displays a "No courses found" message.<br>
            5. User can refine the search by entering a different keyword.<br>
            6. Upon selecting a course, the user is redirected to the course details page.
        </td>
    </tr>
    <tr>
        <th>Outcome</th>
        <td>User finds and accesses the desired course information.</td>
    </tr>
