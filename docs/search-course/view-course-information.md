        <h2>Course Information</h2>  
<p>The Course Information feature provides users with detailed insights into a selected course, including its description, learning outcomes, mentors, and schedule.</p>

<h2>Use Case Scenario</h2>  
<table border="1">
    <tr>
        <th>Use Case</th>
        <th>Course Information</th>
    </tr>
    <tr>
        <th>Actor(s)</th>
        <td>Mentee</td>
    </tr>
    <tr>
        <th>Goal</th>
        <td>View course details, including description, learning outcomes, mentors, and enrollment options.</td>
    </tr>
    <tr>
        <th>Preconditions</th>
        <td>User must be a registered mentee and navigate to a specific course page.</td>
    </tr>
    <tr>
        <th>Main Scenario</th>
        <td>
            1. User selects a course from the list of available courses.<br>
            2. The system displays course details, including title, image, duration, and available mentors.<br>
            3. The user reads the course description and learning outcomes.<br>
            4. The user views the list of available mentors and their schedules.<br>
            5. The user selects a mentor if required.<br>
            6. The user clicks on the "Enroll this Course" button to register.<br>
            7. The system confirms the enrollment and updates the user’s dashboard.
        </td>
    </tr>
    <tr>
        <th>Outcome</th>
        <td>User successfully enrolls in the course and gains access to course materials.</td>
    </tr>
</table>
