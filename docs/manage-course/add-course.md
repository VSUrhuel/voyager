        <h2>Add Course</h2>  
        <p>This is the page where courses are created. This enables the admin to add a new course for the mentorship.</p>
        <h2>Use Case Scenario</h2> 
        <table border="1">
    <tr>
        <th>Use Case</th>
        <th>Adding of Course</th>
    </tr>
    <tr>
        <th>Actor(s)</th>
        <td>Admin</td>
    </tr>
    <tr>
        <th>Goal</th>
        <td>Create or add a new course for the mentorship.</td>
    </tr>
    <tr>
        <th>Preconditions</th>
        <td>The user logged-in must have an admin access.</td>
    </tr>
    <tr>
        <th>Main Scenario</th>
        <td>
            1. Admin clicked on the "+" button on the right side beside the archived button.<br>
            2. Admin will be prompted with a form to input necessary course details.<br>
            3. By clicking "Add Course" with complete filled up form, the course will be successfully created.<br>
        </td>
    </tr>
    <tr>
        <th>Outcome</th>
        <td>The admin was able to create a new course necessary for adaption to changes in the field of study for mentorship</td>
    </tr>
