        <h2>Add Mentor</h2>  
        <p>This page displays an input form that asks for the mentor information. This enables the admin to create a new mentor.</p>
        <h2>Use Case Scenario</h2> 
        <table border="1">
    <tr>
        <th>Use Case</th>
        <th>Adding of Mentor</th>
    </tr>
    <tr>
        <th>Actor(s)</th>
        <td>Admin</td>
    </tr>
    <tr>
        <th>Goal</th>
        <td>Fill-up the form to create or add a new mentor in the application.</td>
    </tr>
    <tr>
        <th>Preconditions</th>
        <td>The user logged in must have an admin access.</td>
    </tr>
    <tr>
        <th>Main Scenario</th>
        <td>
            1. Admin clicked on the "+" button on the right side beside the suspended button to be redirected in this page.<br>
            2. As soon as the admin enters this page, a form for adding mentor will be displayed to fill-up.<br>
            2. After filling-up the form and clicked "Add Mentor", the new mentor is now created and is ready to use.<br>
        </td>
    </tr>
    <tr>
        <th>Outcome</th>
        <td>The admin can now be able to add mentors as much as needed, this would ensure the continuity of the mentorship service in this application</td>
    </tr>
</table>
