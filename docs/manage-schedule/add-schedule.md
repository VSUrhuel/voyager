    <h2>Adding a Schedule</h2>  
    <p>
        The Add Schedule feature allows mentors to set up new mentoring sessions. 
        Mentors can select a date, time, and duration involved. 
        This feature integrates with Google Calendar to provide notifications and reminders.
    </p>    
    <p><strong>Note:</strong> The actual UI design for adding a schedule is still in development and will align with Flutter's built-in calendar system and Google API integration.</p>  
    <h2>Use Case Scenario</h2> 
    <table border="1">
        <tr>
            <th>Use Case</th>
            <th>Adding a New Mentoring Schedule</th>
        </tr>
        <tr>
            <th>Actor(s)</th>
            <td>Mentor</td>
        </tr>
        <tr>
            <th>Goal</th>
            <td>Schedule a new mentoring session with mentees and receive reminders.</td>
        </tr>
        <tr>
            <th>Preconditions</th>
            <td>The mentor is logged into the platform and has access to scheduling features.</td>
        </tr>
        <tr>
            <th>Main Scenario</th>
            <td>
                1. Mentor logs into the platform.<br>
                2. Mentor navigates to the "Add Schedule" section.<br>
                3. Mentor selects a date and time for the session.<br>
                4. Mentor specifies the session duration.<br>
                5. Mentor optionally adds session details (e.g., topics to be covered).<br>
                6. Mentor receives a confirmation notification, and mentees are notified.<br>
            </td>
        </tr>
        <tr>
            <th>Outcome</th>
            <td>The mentoring session is successfully scheduled.</td>
        </tr>
