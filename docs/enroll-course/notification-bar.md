        <h2>Notification Bar (Popup)</h2>  
<p>The notification bar is a temporary popup that appears on any screen to alert users of important updates, reminders, or announcements in real-time. It ensures users stay informed without navigating away from their current activity.</p>

<h2>Use Case Scenario</h2> 
<table border="1">
    <tr>
        <th>Use Case</th>
        <th>Real-time Notification Popup</th>
    </tr>
    <tr>
        <th>Actor(s)</th>
        <td>Mentee</td>
    </tr>
    <tr>
        <th>Goal</th>
        <td>Receive immediate updates about mentorship sessions, quizzes, or other important announcements without switching screens.</td>
    </tr>
    <tr>
        <th>Preconditions</th>
        <td>The user is actively using the app, and a new notification is triggered by the system.</td>
    </tr>
    <tr>
        <th>Main Scenario</th>
        <td>
            1. The user is interacting with any page within the app.<br>
            2. A new notification (e.g., session reminder, quiz update) is triggered.<br>
            3. A small popup bar appears at the top or bottom of the screen with a brief message.<br>
            4. The user can either tap on the notification to view more details or let it fade away after a few seconds.<br>
        </td>
    </tr>
    <tr>
        <th>Outcome</th>
        <td>The user is immediately informed of important updates without interrupting their current activity.</td>
    </tr>
