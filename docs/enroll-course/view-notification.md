        <h2>Notifications</h2>  
<p>This page displays important updates, reminders, and announcements for the user. Notifications may include upcoming mentorship sessions, study reminders, and other relevant alerts. The page categorizes notifications into "Today" for recent updates and "Previous Notifications" for older messages.</p>

<h2>Use Case Scenario</h2> 
<table border="1">
    <tr>
        <th>Use Case</th>
        <th>View Notifications</th>
    </tr>
    <tr>
        <th>Actor(s)</th>
        <td>Mentee</td>
    </tr>
    <tr>
        <th>Goal</th>
        <td>Stay informed about mentorship sessions, study materials, and other updates</td>
    </tr>
    <tr>
        <th>Preconditions</th>
        <td>User must have a registered mentee account and enrolled in at least one course</td>
    </tr>
    <tr>
        <th>Main Scenario</th>
        <td>
            The user opens the notifications page.<br>
            The user views the most recent notifications under "Today".<br>
            The user checks older notifications under "Previous Notifications".<br>
            The user clicks on a notification to access further details if necessary.<br>
        </td>
    </tr>
    <tr>
        <th>Outcome</th>
        <td>The user stays updated with upcoming mentorship sessions and important academic reminders.</td>
    </tr>
