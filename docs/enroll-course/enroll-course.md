        <h2>Enroll Course</h2>  
<p>This page provides an overview of the selected course, including its duration, available mentors, and mentees. Users can read the course description, review the learning outcomes, and select a mentor before enrolling. The "Enroll this Course" button at the bottom allows users to confirm their participation and begin their mentorship journey.</p>

            The user reviews the course details, including description and learning outcomes.<br>

            The user clicks the "Enroll this Course" button.<br>
        </td>
    </tr>
