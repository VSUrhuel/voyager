        <h2>Login with Google</h2>  
    <p>
        The Login with Google feature allows users to quickly and securely access their accounts using 
        their Google credentials. This eliminates the need to remember passwords and enhances user experience 
        with a seamless authentication process.
    </p>
    <h2>Use Case Scenario</h2> 
    <table border="1">
        <tr>
            <th>Use Case</th>
            <th>Google Authentication</th>
        </tr>
        <tr>
            <th>Actor(s)</th>
            <td>Registered User</td>
        </tr>
        <tr>
            <th>Goal</th>
            <td>Log in to the platform using Google authentication.</td>
        </tr>
        <tr>
            <th>Preconditions</th>
            <td>User must have a valid Google account.</td>
        </tr>
        <tr>
            <th>Main Scenario</th>
            <td>
                1. User navigates to the login page.<br>
                2. User selects "Login with Google".<br>
                3. System redirects to Google authentication.<br>
                4. User selects their Google account and grants necessary permissions.<br>
                5. If authentication is successful, the system creates/logs in the user.<br>
                6. User is redirected to the dashboard.<br>
            </td>
        </tr>
        <tr>
            <th>Outcome</th>
            <td>User successfully logs in using Google and gains access to the platform.</td>
        </tr>
    </table>
