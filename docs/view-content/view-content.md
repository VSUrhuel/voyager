        <h2>View Content</h2>  
<p>The View Content page allows users to access important learning materials, announcements, and updates related to their courses. It provides categorized content based on recent and past updates.</p>

<h2>Use Case Scenario</h2>  
<table border="1">
    <tr>
        <th>Use Case</th>
        <th>View Course Content</th>
    </tr>
    <tr>
        <th>Actor(s)</th>
        <td>Mentee</td>
    </tr>
    <tr>
        <th>Goal</th>
        <td>Allow students to access and review course-related content, including announcements, materials, and study resources.</td>
    </tr>
    <tr>
        <th>Preconditions</th>
        <td>The user must be enrolled in a course and have access to the learning platform.</td>
    </tr>
    <tr>
        <th>Main Scenario</th>
        <td>
            1. User navigates to the View Content page.<br>
            2. The system displays recent course-related content." <br>
            3. Mentee can read and follow instruction posted by the mentor" <br>
            4. If there are no new updates, a "No more content" message is displayed.<br>
        </td>
    </tr>
    <tr>
        <th>Outcome</th>
        <td>The student successfully views course materials and stays updated on relevant learning resources.</td>
    </tr>
