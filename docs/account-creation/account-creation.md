        <h2>Account Creation</h2>  
    <p>Mentees can create a new account to access the platform's features. This step involves providing necessary details such as student ID, name, email, and password. </p>
    <h2>Use Case Scenario</h2> 
    <table border="1">
        <tr>
            <th>Use Case</th>
            <th>Account Creation</th>
        </tr>
        <tr>
            <th>Actor(s)</th>
            <td>Mentee</td>
        </tr>
        <tr>
            <th>Goal</th>
            <td>Create a new account to access the platform.</td>
        </tr>
        <tr>
            <th>Preconditions</th>
            <td>The user does not have an existing account and needs to register.</td>
        </tr>
        <tr>
            <th>Main Scenario</th>
            <td>
                1. User navigates to the registration page.<br>
                2. User enters required information (e.g., name, email, password).<br>
                3. User submits the form.<br>
                4. System validates the input (e.g., strong password, and valid email).<br>
            </td>
        </tr>
        <tr>
            <th>Outcome</th>
            <td>The user successfully creates an account and can proceed to <span><a href="https://github.com/VSUrhuel/voyager/blob/main/docs/account-creation/email-verification.md">email verfication</span>.</td>
