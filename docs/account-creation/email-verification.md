    <h2>Email Verification</h2>  
    <p>After signing up, users must verify their email to activate their account. This step ensures security and prevents unauthorized access.</p>
    <h2>Use Case Scenario</h2> 
    <table border="1">
        <tr>
            <th>Use Case</th>
            <th>Email Verification</th>
        </tr>
        <tr>
            <th>Actor(s)</th>
            <td>Mentee</td>
        </tr>
        <tr>
            <th>Goal</th>
            <td>Verify email to activate the account.</td>
        </tr>
        <tr>
            <th>Preconditions</th>
            <td>The user has signed up but has not yet verified their email.</td>
        </tr>
        <tr>
            <th>Main Scenario</th>
            <td>
                1. User receives a verification email.<br>
                2. User clicks the verification link.<br>
                3. System validates the link and activates the account.<br>
                4. User gains access to the platform.<br>
            </td>
        </tr>
        <tr>
            <th>Outcome</th>
            <td>The user's email is verified, and they can log in.</td>
        </tr>
