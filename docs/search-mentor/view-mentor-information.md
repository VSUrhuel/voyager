        <h2>Mentor Information</h2>  
<p>The Mentor Information page provides detailed information about a selected mentor, including their name, contact details, experience, languages spoken, and completed mentorship sessions. Users can review a mentor's profile before deciding to connect with them.</p>

<h2>Use Case Scenario</h2>  
<table border="1">
    <tr>
        <th>Use Case</th>
        <th>Mentor Profile</th>
    </tr>
    <tr>
        <th>Actor(s)</th>
        <td>Mentee</td>
    </tr>
    <tr>
        <th>Goal</th>
        <td>Allow users to view a mentor's profile and learn about their experience, background, and expertise.</td>
    </tr>
    <tr>
        <th>Preconditions</th>
        <td>User must navigate to a mentor’s profile from the mentors listing page.</td>
    </tr>
    <tr>
        <th>Main Scenario</th>
        <td>
            1. User selects a mentor from the mentors listing page.<br>
            2. The system displays the mentor’s profile with their name, photo, and contact information.<br>
            3. User can read the "About" section to learn more about the mentor.<br>
            4. User can review the mentor’s work experience and completed mentorship sessions.<br>
            5. The mentor’s known languages are displayed, allowing users to identify language compatibility.<br>
            6. Social media links are provided for further connection.<br>
            7. User can choose to proceed with mentorship or contact the mentor for more details.
        </td>
    </tr>
    <tr>
        <th>Outcome</th>
        <td>User successfully reviews a mentor’s profile and makes an informed decision about mentorship.</td>
    </tr>
